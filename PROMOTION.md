## 🎯 客服话术 (常见问题回复)

**Q: 和直接用 DeepSeek 官方 API 有什么区别？**
A: 我们封装了场景化端点，你不需要写 Prompt 模板。注册即用，不需要绑定海外信用卡。按量付费，¥10就能开始用。

**Q: 数据安全吗？**
A: 所有 API Key 加密存储，HTTPS 传输。我们不存储你的调用内容，只记录 token 用量用于计费。

**Q: 支持开发票吗？**
A: 目前仅支持个人用户，暂不开发票。企业客户可联系客服协商。

**Q: 点数会过期吗？**
A: 永久有效，不会过期。

**Q: 可以退款吗？**
A: 未使用的点数支持退款（扣除手续费用）。已消耗的点数不支持退款。
