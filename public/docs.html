简介

DeepSeek API Service 封装了 DeepSeek 大模型能力，提供开箱即用的 AI 写作、中英翻译、文本摘要、Chat 对话、代码生成、代码审查、数据分析 七大垂直场景 API。用户按点数付费，注册即送 100 点。

Base URL: http://localhost:3456

快速开始

1. 获取邀请码

本服务采用邀请制注册，请联系管理员获取邀请码。

2. 注册账号

curl -X POST http://localhost:3456/api/register \
  -H "Content-Type: application/json" \
  -d '{"username": "myapp", "password": "mypassword", "invite_code": "XXXXXXXX"}'

返回 API Key 和初始 100 点数：

{
  "success": true,
  "data": {
    "username": "myapp",
    "api_key": "dsk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    "balance": 100
  }
}

3. 调用 API

在请求头中携带 API Key：

curl -X POST http://localhost:3456/api/writer \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer dsk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -d '{"input": "写一篇关于人工智能的科普文章，500字"}'

4. 查看余额

curl http://localhost:3456/api/me \
  -H "Authorization: Bearer dsk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
💡 新用户注册即送 100 点，可以免费试用。

认证方式

所有需要认证的 API 端点使用 Bearer Token 认证。在 HTTP 请求头中添加：

Authorization: Bearer <your-api-key>

API Key 格式为 dsk- 开头 + 32 位十六进制字符。

⚠️ 请妥善保管您的 API Key。不要在客户端代码中暴露，建议在后端服务中使用。

AI 写作

POST /api/writer

调用 DeepSeek 大模型生成各类中文内容：文章、营销文案、社交媒体内容、邮件、演讲稿等。

请求参数

参数类型必填说明
inputstring是写作需求描述，最多 50000 字符
max_tokensnumber否最大输出长度，默认 2048，最大 8192
curl -X POST http://localhost:3456/api/writer \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "写一篇关于春天的散文，300字左右"}'

响应示例：

{
  "success": true,
  "data": {
    "result": "春天来了，万物复苏...",
    "usage": {
      "prompt_tokens": 45,
      "completion_tokens": 320,
      "points_cost": 1,
      "balance_remaining": 99
    }
  }
}

中英翻译

POST /api/translator

自动识别中英文方向，提供专业级翻译。中文翻译符合中文习惯，英文翻译地道流畅。

请求参数

参数类型必填说明
inputstring是需要翻译的文本
max_tokensnumber否最大输出长度，默认 2048
curl -X POST http://localhost:3456/api/translator \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "Artificial intelligence is transforming every industry."}'
{
  "success": true,
  "data": {
    "result": "人工智能正在改变每一个行业。",
    "usage": {
      "prompt_tokens": 20,
      "completion_tokens": 15,
      "points_cost": 1,
      "balance_remaining": 98
    }
  }
}

文本摘要

POST /api/summary

将长文本提炼为精炼摘要，默认输出 3 种长度：一句话总结、段落摘要、详细摘要。

curl -X POST http://localhost:3456/api/summary \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "（长文本内容...）"}'

Chat 对话

POST /api/chat

多轮对话接口，支持完整的 messages 数组，适用于需要上下文的连续对话场景。

请求参数

参数类型必填说明
messagesarray是消息数组，每项含 role (user/assistant/system) 和 content，最多50条
max_tokensnumber否最大输出长度，默认 2048，最大 8192
temperaturenumber否温度参数 0-2，默认 0.7
curl -X POST http://localhost:3456/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"messages": [{"role": "user", "content": "你好，帮我看看这个问题"}]}'

代码生成

POST /api/codegen

根据自然语言需求生成高质量代码，支持多种编程语言，包含注释和最佳实践。

请求参数

参数类型必填说明
inputstring是编程需求描述，最多 50000 字符
languagestring否目标语言，默认 auto（自动判断）
max_tokensnumber否最大输出长度，默认 4096
curl -X POST http://localhost:3456/api/codegen \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "写一个 Python 函数，实现快速排序", "language": "python"}'

代码审查

POST /api/codereview

AI 代码审查，检测 bug、安全漏洞、性能问题和可维护性问题，按严重程度排序。

curl -X POST http://localhost:3456/api/codereview \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "function foo(x) { eval(x); }"}'

数据分析

POST /api/dataanalysis

对输入数据进行统计分析，发现趋势、模式和异常，提供可操作的建议。

curl -X POST http://localhost:3456/api/dataanalysis \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"input": "月份: 1月,2月,3月,4月\n销售额: 120,135,148,162"}'

账户信息

查询当前 API Key 的余额和使用统计。

curl http://localhost:3456/api/me \
  -H "Authorization: Bearer $API_KEY"
{
  "success": true,
  "data": {
    "username": "myapp",
    "balance": 95,
    "usage_stats": [
      { "endpoint": "writer", "calls": 3, "total_cost": 3 },
      { "endpoint": "translator", "calls": 1, "total_cost": 1 }
    ]
  }
}

账号注册

POST /api/register

邀请制注册。用户名 3-32 字符，密码 6-128 字符，需要有效邀请码。

请求参数

参数类型必填说明
usernamestring是用户名，3-32 字符
passwordstring是密码，6-128 字符
invite_codestring是邀请码，联系管理员获取
curl -X POST http://localhost:3456/api/register \
  -H "Content-Type: application/json" \
  -d '{"username": "newuser", "password": "securepass", "invite_code": "XXXXXXXX"}'
🎁 新注册用户自动赠送 100 点。邀请码由管理员在后台生成。

套餐查询

GET /api/payments/packages

获取可用的充值套餐和定价，无需认证。

创建充值订单

POST /api/payments/create-order

选择套餐创建支付订单，返回二维码链接。需要认证。

参数类型必填说明
package_idstring是套餐ID：basic / starter / popular / pro / ultimate
curl -X POST http://localhost:3456/api/payments/create-order \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"package_id": "popular"}'

订单查询

GET /api/payments/order/:order_no GET /api/payments/orders

查询单个订单状态或订单历史列表。

定价说明

套餐价格点数赠送性价比
基础包¥101,000-1元=100点
入门包¥505,000+5001元=110点
人气包¥10010,000+2,0001元=120点
专业包¥20020,000+5,0001元=125点
企业包¥50050,000+15,0001元=130点

计费规则： 每 1000 个输出 token 约消耗 1 点（约 ¥0.01 成本）。实际扣费基于 DeepSeek 官方的 prompt_tokens 和 completion_tokens 计算，约 10 倍加价。

速率限制

端点类型限制时间窗口
AI 端点 (writer/translator/summary)30 次每分钟
认证端点 (register/login/me)10 次每分钟
管理端点 (admin/*)5 次每分钟

超出限制后返回 HTTP 429 Too Many Requests。

错误码

HTTP 状态码错误信息说明
400请求参数无效缺少必填字段或格式不正确
401缺少/无效的 API Key未提供 Authorization header 或 Key 无效
402余额不足点数不足以完成本次请求
403管理员密码错误X-Admin-Key header 不正确
404用户/订单不存在请求的资源未找到
429请求过于频繁超过速率限制，请稍后再试
500服务暂时不可用服务器内部错误

所有错误响应格式统一为：

{"success": false, "error": "错误描述"}
